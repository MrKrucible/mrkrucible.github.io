mumble_setup

Mumble is a free voice chat application that uses military-grade encryption. Even with encryption, Mumble has the lowest latency out of any VoIP app. This makes it the voip of choice for professional competitive gamers. Even the mobile versions (Plumble & Mumblefy) will use minimal amounts of bandwidth @10-40kb/packet.

1. Download

Get the latest snapshot of Mumble for your system.

The snapshots currently (July 18th, 2015) have an overlay that works for 64-bit games, and stable versions don't. Once you have Mumble installed execute the wizard and complete the install; and install the mumble package on linux machines.

2. Connecting

You can either click here to auto-connect:


(once you have Mumble installed)

Or, you can connect by entering the server info. It can be any Mumble's info, but we're gonna use mine.

Click on the globe icon to the top left. Click on Add New... and new window will appear. You will be asked for our server info below.

  • Label: JustSomeGamers
  • Address: mumble.justsomegamers.com
  • Port: 6052
  • Username: Use a screen name

If you are using the auto-connect button, you can save the server info by clicking on the globe while connected to the server; then click on the Add New... button. It will auto fill the info and it'll be added to your favorites for later!
